# Quick Start Guide

## Rapid Deployment to Railway
1. Visit [Railway](https://railway.app).
2. Create a new project or use an existing one.
3. Link your GitHub repository to Railway.
4. Configure the environment variables as needed.
5. Click on the Deploy button to start the deployment.

## Local Testing
1. Clone the repository:
   ```bash
   git clone https://github.com/neepak100/ai-dropship-system.git
   cd ai-dropship-system
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000` to view the application.
